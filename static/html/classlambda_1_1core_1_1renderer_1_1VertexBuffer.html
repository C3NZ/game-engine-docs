Creates a VertexBuffer through the Graphics API that is being used at compile time.

Parameters
vertices- a pointer to an array of vertices to be registered.
size- The size of the vertices in bytes. This is the primary method of creating platform independent vertex buffers and what should be used by users to create Vertex Buffers that are compatible with the rendering API.

◆ Unbind()

VertexBuffer::Unbind ( ) const
pure virtual

Unbind the vertex buffer to the current rendering context.

The unbinding process is entirely dependent upon the grahpics API. However, it should be noted that this function needs to be called by the user whenever they're trying to unbind the buffer from the render API. (For cleaning up anything still in memory.)

Implemented in lambda::platform::opengl::OpenGLVertexBuffer.
