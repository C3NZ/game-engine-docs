void lambda::core::imgui::ImGuiLayer::OnAttach ( )
overridevirtual

What to do when attached to lambda.

The default OnAttach creates a context, activates dark mode, and register inputs with GLFW keys. This implementation currently only supports OpenGL and will be modified in the future to use engine specified key codes.
