Detailed Description

Buffer abstractions that allow the ease of implementing Buffers for any graphics API.

All platform specific graphics API will implement buffer implementations in their corresponding API syntax through this generalized engine API. This will allow application developers to not have to worry (Not entirely true) about platform specific buffer implementations.
