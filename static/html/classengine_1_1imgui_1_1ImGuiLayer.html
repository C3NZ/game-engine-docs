◆ OnAttach()

engine::imgui::ImGuiLayer::OnAttach ( )
overridevirtual

Handles an ImGuiLayers attachment to the engine.

The default OnAttach creates a context, activates dark mode, and register inputs with GLFW keys. This implementation currently only supports OpenGL and will be modified in the future to use engine specified key codes.

This is currently setup with a default implementation but will most likely be delegated to users that would like to implement their own imgui layers.
