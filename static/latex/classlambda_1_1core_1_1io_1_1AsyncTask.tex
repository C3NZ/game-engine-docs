\doxysubsection{Detailed Description}
A wrapper for callbacks that are supposed to be executed asynchronously. 

There is no need to use this class externally, as the \mbox{\hyperlink{classlambda_1_1core_1_1io_1_1EventLoop}{Event\+Loop}} will create these upon passing it a Async\+Callback.
