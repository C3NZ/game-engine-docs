\doxysubsection{Detailed Description}
Asynchronous Event Loop that allows the execution of code to happen in another thread. This is not recommended for production as of yet. 

This currently depends on everything passed into the queue being atomic or protected by locks to ensure that data isn\textquotesingle{}t corrupted. The easy way to remedy this is to copy data into your callback as opposed to using instances of it, as that has the potential for major issues.
