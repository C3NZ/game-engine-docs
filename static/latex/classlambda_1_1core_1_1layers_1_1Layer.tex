\doxysubsection{Detailed Description}
The lambda \mbox{\hyperlink{classlambda_1_1core_1_1layers_1_1Layer}{Layer}} abstraction. Primarily used for direct access into lambdas tick and event system. 

Anyone implementing lambda as a library in their project will at some point need to create a layer in order to receive updates and events from lambda.
