\doxysubsection{Detailed Description}
Buffer abstractions that allow the ease of implementing Buffers for any graphics A\+PI. 

All platform specific graphics A\+PI will implement buffer implementations in their corresponding A\+PI syntax through this generalized engine A\+PI. This will allow application developers to not have to worry (Not entirely true) about platform specific buffer implementations.
