\doxysubsection{Detailed Description}
The event handling system. 

The \mbox{\hyperlink{classengine_1_1events_1_1EventDispatcher}{Event\+Dispatcher}} is the key to handling all events. It is created per event and streamlines the process of dispatching events with callbacks that are invoked when the Event\+Type of the event used to create the \mbox{\hyperlink{classengine_1_1events_1_1EventDispatcher}{Event\+Dispatcher}} matches the Event\+Type that the callback is looking for.
