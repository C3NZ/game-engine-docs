\doxysubsection{Detailed Description}
The base \mbox{\hyperlink{classengine_1_1renderer_1_1VertexBuffer}{Vertex\+Buffer}} class to be used for creating vertex buffers. 

Platform specific graphics A\+PI should extend this class in order to be supported by the the rendering A\+PI.
