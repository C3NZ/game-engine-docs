\doxysubsection{Detailed Description}
The Shader A\+PI. 

Expects sources to be strings that are designed for the shading language that is being used in the A\+PI. This can also load shaders stored within a glsl file and conforms to the way that the Shader A\+PI expects to read them. Look into the sandbox tool for working examples.
